正在做日綜／日本搞笑藝人字幕的日文→台灣繁中翻譯。目標不是正式學術翻譯，而是讓中文字幕自然、好讀、辨識度高，同時不要亂創台式譯名。

## 專名翻譯總原則

專名包含：節目名、單元名、搞笑團體名、藝人名、藝名、作品名、品牌名、梗名。

整體優先順序：

1. **已有穩定台灣／中文譯名就用既有譯名**
   - 只在能確認常見、穩定、官方或中文圈已廣泛使用時採用。
   - 不要自己發明看起來很像台灣節目名的「創意譯名」。
   - 例：
     - `ロンドンハーツ` → `男女糾察隊`
     - `ロンハー` → `男女糾察隊`
     - `逃走中` → `全員逃走中`
     - `るろ剣` → `神劍闖江湖`
     - `南海キャンディーズ` → `南海甜心`
     - `ネプチューン` → `海王星`

2. **人名／藝名可回推漢字時，要看該漢字是不是名稱核心**
   - 若假名只是本名、姓氏、名字的假名表記，或中文圈已有穩定漢字用法，可用繁中漢字。
     - `河井ゆずる` / 本名 `河井譲` → `河井讓`
     - `お見送り芸人しんいち` / 本名 `上野晋一` → `送別藝人晉一`
     - `イワクラ` / 原本表記 `岩倉美里` → `岩倉`
     - `月亭方正` / `つきていほうせい` → `月亭方正`

   - 若假名部分可明確回推為姓名漢字，而且該漢字本身是藝名來源、梗點或辨識重點，應保留漢字；其餘外來語／假名部分再用英文或羅馬字。
     - `きむらバンド` → `木村Band`
     - 不建議 → `Kimura Band`
     - 不建議 → `木村樂團`

   - 但若藝名是刻意片假名化、暱稱化、角色化，即使知道本名漢字，也不一定回推成本名漢字；通常保留藝名讀音，用羅馬字。
     - `ユースケ` → `Yusuke`，不必回推成 `裕介`
     - `ガク` → `Gaku`
     - `ケツ` → `Ketsu`
     - `きょん` → `Kyon`，不擅自補成 `恭`
     - `おたけ` → `Otake`，不擅自補成 `小武` 或 `阿武`

3. **語意清楚、翻成中文後仍像專名時，才翻中文**
   - 可以翻節目名、單元名、團名、藝名中的描述性成分。
   - 但如果翻完會變成普通名詞、食品名、物品名、普通句子、店名，導致不像藝人／團體名，就不要硬翻，改用羅馬字或既有英文表記。
   - 可翻例：
     - `熊元プロレス` → `熊元摔角`
     - `紅しょうが` → `紅生姜`
     - `ママタルト` → `媽媽塔`
     - `チャンスの時間` → `機會的時間`
     - `囲碁将棋` → `圍棋將棋`
     - `いろはに千鳥` → `伊呂波千鳥`
   - 不建議直譯例：
     - `見取り図` 不建議 → `示意圖`；建議 → `Mitorizu`
     - `銀シャリ` 不建議 → `銀舍利`；建議 → `銀Shari`
     - `モグライダー` 不建議 → `鼴鼠騎士`；建議 → `Mogu Rider`
     - `おいでやす小田` 不建議 → `歡迎光臨小田`；建議 → `Oideyasu小田`
     - `ロングコートダディ` 不建議 → `長袍老爹`；建議 → `Long Coat Daddy`
     - `カナメストーン` 不建議 → `要石`；建議 → `Kaname Stone`
     - `サルゴリラ` 不建議 → `猴猩猩`；建議 → `Sarugorilla`
     - `アンタッチャブル` 不建議 → `不可觸碰`；建議 → `Untouchable`

4. **純假名／片假名藝名、團名，沒有可回推漢字或清楚語意時，優先羅馬字**
   - 中文字幕裡不要隨便保留日文假名，除非保留假名真的比較有辨識度。
   - 如果原名是「漢字 + 假名／片假名」混合表記，通常保留有辨識度的漢字部分，假名部分羅馬字化；但若已有官方／常見英文表記，或保留漢字會讓中文看起來像自創混搭名，則優先使用整體羅馬字。
   - 例：
     - `松井ケムリ` → `松井Kemuri`
     - `真中まな` → `真中Mana`
     - `鈴木もぐら` → `鈴木Mogura`
     - `今井らいぱち` → `今井Raipachi`
     - `銀シャリ` → `銀Shari`
     - `や団` → `Yadan`
     - `なえなの` → `naenano`
     - `ほんこん` / `ホンコン` → `Honkon`
     - `べーやん` / `ベーヤン` → `Beeyan`

5. **羅馬字優先用官方／常見英文表記，不要機械逐音轉寫**
   - 若已有官方、常見、資料庫慣用英文表記，優先採用該表記。
   - 英文表記要盡量固定大小寫與空格。
   - 同一組合在節目名裡也要維持一致。
   - 例：
     - `ダウ90000` → `Daw90000`
     - `かもめんたる` → `Kamomental`
     - `パンプキンポテトフライ` → `Pumpkin Potato Fry`
     - `ダウンタウン` → `DOWNTOWN`
     - `ダウンタウンDX` → `DOWNTOWN DX`
     - `水曜日のダウンタウン` → `星期三的DOWNTOWN`
     - `さまぁ〜ず` → `SUMMERS`
   - 不要只照日文音節硬拼出不自然的羅馬字。

6. **保留日文原文，尤其是假名／片假名，應該是最後手段**
   - 只有在翻中文和羅馬字都會降低辨識度，或該名稱是明顯的日文文字遊戲／節目標題時，才保留。
   - 例：
     - `かまいガチ` 可保留 `かまいガチ`
   - 但不要把以下名稱預設保留成假名：
     - `カカロニ`
     - `ケムリ`
     - `すがちゃん`
     - `や団`
     - `なえなの`
     - `ほんこん`

## 專名翻譯和一般字幕翻譯要分開

一般字幕句子中，不要擅自補「我／你／他／她／我們／大家」等日文省略主詞。

但節目名、藝名、梗名是固定名詞，可以為了保留語氣做固定化翻譯。

例：

- 一般句子裡，`知らんけど` 不一定要補成「我是不知道啦」。
- 但節目名 `かまいたちの知らんけど` 可以翻成 `鎌鼬的我是不知道啦`，因為這是在翻固定節目名／口頭禪語感，不是普通句子補主詞。

## jp array 的使用原則

`jp` 陣列可以收多種表記，目的是吸收 ASR 錯字、平假名／片假名差異、長音、空格、驚嘆號、全半形差異。

可以加入：
- 正式表記
- 常見異體
- ASR 可能誤轉表記
- 節目字幕可能出現的表記
- 有無空格、全形／半形差異

例：

```json
{
  "jp": ["アメトーク", "アメトーーク", "アメトーーク!"],
  "zh": "Ametalk"
}
````

```json
{
  "jp": ["熊元 プロレス", "熊元　プロレス", "熊元プロレス"],
  "zh": "熊元摔角"
}
```

```json
{
  "jp": ["ほんこん", "ホンコン"],
  "zh": "Honkon"
}
```

但不要把真正可能是不同意思的詞硬塞同一組。

壞例：

```json
{
  "jp": ["ホンコン", "香港"],
  "zh": "Honkon"
}
```

因為 `香港` 漢字出現時很可能真的是地名香港。

## 不要做的事

* 不要把片假名藝名直接照意思硬翻。

  * 壞例：`松井ケムリ` → `松井煙`
  * 好例：`松井ケムリ` → `松井Kemuri`

* 不要擅自幫假名人名補漢字。

  * 壞例：`きょん` → `恭`
  * 好例：`きょん` → `Kyon`

* 不要把節目名翻成很像台灣節目的創意標題。

  * 壞例：`かまいガチ` → `鐮鼬真挑戰`
  * 好例：`かまいガチ` → `かまいガチ`

* 不要把搞笑團體名硬翻成普通名詞。

  * 壞例：`見取り図` → `示意圖`
  * 好例：`見取り図` → `Mitorizu`

* 不要把字面有意思的藝名翻成普通句子。

  * 壞例：`おいでやす小田` → `歡迎光臨小田`
  * 好例：`おいでやす小田` → `Oideyasu小田`

* 不要把暱稱化藝名翻成自創中文暱稱。

  * 壞例：`おたけ` → `小武` / `阿武`
  * 好例：`おたけ` → `Otake`

* 不要在中文字幕裡大量留下日文假名。

  * 假名會讓中文觀眾閱讀斷裂。
  * 翻成中文後仍像專名，才中文化；否則優先羅馬字／常見英文表記；保留假名放最後。
